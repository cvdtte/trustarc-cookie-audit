How to identify the CMP

Three ways to detect:

  1. Elements tab — Ctrl+F → search cookie, consent, banner. Look at IDs/classes.
  2. Console probe — paste window.OneTrust, window.truste, window.Cookiebot, window.UC_UI, window.Didomi, window.Osano, window.__tcfapi, window.__gpp — whichever returns a defined object identifies the CMP.
  3. Network tab — filter by Domain → look for cookielaw.org, trustarc.com, cookiebot.com, usercentrics.eu, didomi.io, sp-prod.net, quantcast.com, osano.com, etc.
